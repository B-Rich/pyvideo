About this site

This is a video index of Python-related video on the Internet. It does not host video—it's an index.

Each video is lovingly packaged with a variety of metadata that comprises the context of the video: When was it taken? What was the venus? Who was speaking? What was the topic of the talk? What are the related links? Are there quality issues with the video?

It's my hope that this metadata brings the video that's currently scattered about easier to find and easier to use.

Contact me

If you've found a bug in richard, then please write up an issue in the issue tracker.

If you want to contribute to this site, you'll have to wait a bit until we get a contributor infrastructure going. I really want to make it easy for people to help make the metadata better, but the code isn't there, yet.

In the meantime, if you have questions, comments or concerns about this site, want to help out with richard development, or just want to send encouraging words, then send an email to willg at bluesock dot org.

How?

The software tha truns this site is called richard. It's a free software project. The code is all on Github at http://github.com/willkg/richard.

It's currently using:

  • django
  • django-nose/nose
  • django-haystack and whoosh
  • jingo/jinja2
  • Twitter bootstrap
